Pendahuluan
=================================================

Selamat datang di SIDADU.

SIDADU atau Sistem Informasi Data Kependudukan adalah sebuah sistem yang dibangun guna mempermudah proses rekapitulasi data kependudukan desa Binaan Fakultas Ilmu Sosial dan Ilmu Politik UPN "Veteran" Yogyakarta.
Melalui sistem ini, diharapkan nantinya proses pendataan data kependudukan desa binaan dapat berjalan lebih efektif dan efisien.
